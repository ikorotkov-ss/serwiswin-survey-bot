# Oprosnik (Survey Bot) — Telegram-бот опросника SerwisWin

## О проекте
Telegram-бот [@SerwisWinSurveyBot](https://t.me/SerwisWinSurveyBot) для сбора историй, цитат и инсайтов от сотрудников (колл-центр, мастера). Помогает улучшить рекламу и контент.

## Технологии
Python, python-telegram-bot, SQLite, whisper.cpp (расшифровка голоса — опционально, зависит от сервера)

## Репозиторий
`/Users/igorkotkov/_Serwiswin/Oprosnik/` — отдельная папка, не подпроект с CLAUDE.md внутри .claude/

## Ключевые файлы
- `PROJECT_SUMMARY.md` — полная история, архитектура, инструкция деплоя
- `serwiswin_survey_questions.md` — текст опросника (45 вопросов)
- `bot.py` — основной код бота
- `deploy/` — Ansible playbook, systemd unit, setup.sh

## Функционал
- Голосовые сообщения — сохраняются как .ogg, пользователь указывает номер вопроса
- Текстовые ответы — автораспознавание номера вопроса из текста
- Расшифровка голоса через whisper.cpp (опционально, для мощных серверов)
- Разделение по ролям: колл-центр / мастера
- Прогресс-бар, пропуск вопросов, возврат к пропущенным
- `/stats`, `/export`, `/about` (только админы)

## Будущее
После завершения опроса код можно переиспользовать под другие опросники.
