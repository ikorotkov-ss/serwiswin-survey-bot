# SerwisWin Survey Bot — История и контекст проекта

## Идея

SerwisWin — компания по ремонту окон в Варшаве. Нужно улучшить рекламу (FB, IG, Google, TikTok и все каналы) и видео-контент. Для этого решили собрать от сотрудников реальные истории, цитаты и инсайты клиентов — что они говорят, из-за чего переживают, за что благодарят.

## Опросник

Разработан опросник на 45 вопросов, разбитый на блоки:

| Блок | Вопросы | Для кого |
|---|---|---|
| Универсальный | 1-15 | Все |
| Колл-центр | 16-25 | Продавцы / колл-центр |
| Реновация окон | 26-31 | Мастера (только кто делает реновацию) |
| Мастера (выезд) | 32-41 | Мастера |
| Финальный | 42-45 | Все |

Файл опросника: `serwiswin_survey_questions.md`

## Telegram-бот

Создан бот [@SerwisWinSurveyBot](https://t.me/SerwisWinSurveyBot) для сбора ответов.

**Функционал:**
- Приём голосовых сообщений, расшифровка через whisper.cpp (автоопределение языка: русский, польский, украинский, чешский, немецкий)
- Приём текстовых ответов
- Автораспознавание номера вопроса из речи (паттерны: «Вопрос 5», «5.», «номер 5»)
- Если номер не распознан — бот просит уточнить цифрой
- Разделение по ролям: колл-центр / мастера (выбор через кнопки)
- `/stats` — только для админов
- `/export` — выгрузка CSV, только для админов
- `/about` — информация о боте

## Инфраструктура

- **Язык:** Python 3.11
- **Транскрибация:** whisper-cpp (модель `ggml-small`) через локальный Metal GPU на Mac
- **Хранение:** SQLite (`survey.db`)
- **Формат ответов:** голосовые .ogg (сохраняются в `audio/`)
- **Deploy:** будет на сервере программистами через systemd

## Технические детали

- whisper-cpu не найден — используется whisper-cli через subprocess
- Перед транскрибацией .ogg конвертируется в .wav через ffmpeg
- Если whisper не смог расшифровать — бот пишет об этом пользователю
- Для работы whisper обязателен флаг `-l auto` (автоопределение языка)
